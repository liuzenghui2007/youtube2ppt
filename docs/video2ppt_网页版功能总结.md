# 网页版「视频转 PPT」工具功能总结

> 参考对象：video2ppt.com 等在线视频转演示文稿工具  
> 本文档用于记录其功能设计，便于与本项目（本地 YouTube 视频提取 PPT）对比或借鉴。

---

## 一、工具概览

该工具是一个**在线视频内容处理平台**，主要用途：

- 从视频中**自动提取关键帧**（多为 PPT 页面或讲者画面）
- 支持**手动补充、删减、编辑**关键帧
- 最终**导出为演示文稿**（如图片集、PDF、PPTX 等）

适用场景：含 PPT 演示、讲者讲解的课程视频、会议录像等。

---

## 二、核心功能

### 1. 视频上传与管理

- **视频导入**：支持上传本地视频（如 `video_cropped.mp4`）
- **工作空间 (Workspace)**：集中管理当前正在处理的视频

### 2. 智能关键帧提取与预览

| 功能 | 说明 |
|------|------|
| **自动场景检测** | 分析视频，识别画面变化明显的关键帧（对应 PPT 翻页或画面切换） |
| **缩略图网格** | 关键帧以网格展示，每张带时间戳（如 00:00, 00:18） |
| **视频播放器** | 左侧可播放视频，在时间轴上定位，便于对照 |
| **时间轴与帧标记** | 底部时间轴用蓝色竖线标出已提取关键帧位置，方便查看分布 |
| **手动捕获帧** | 拖动时间轴或按「C」键，可手动截取当前帧，弥补自动检测遗漏 |

### 3. 采样设置 (Sampling Settings)

- **智能人像识别**：在有讲者的视频中优化对 PPT 的提取，或纯 PPT 场景下减少误检
- **时间间隔采样**：预设 1s / 5s / 10s 及「自定义」，按翻页快慢调整采样密度
- **重新采样**：修改设置后一键重新提取关键帧

### 4. 关键帧管理

| 操作 | 说明 |
|------|------|
| **单选/取消** | 缩略图上的勾选可切换，只导出选中的帧 |
| **全选/批量** | 如「全选 (23/24)」，显示已选帧数 |
| **编辑单帧（剪刀图标）** | 对单帧裁剪、调整，保证导出页面质量 |
| **删除单帧（垃圾桶）** | 去掉无关或重复帧 |
| **下载单帧** | 单独下载某一张关键帧图片 |

### 5. 导出与分享

- **下载全部**：将当前选中的关键帧打包下载（图片集 / PDF / PPTX 等）
- **分享**：生成分享链接或分享到其他平台

---

## 三、体验特点（可借鉴点）

- **界面**：三栏布局（视频 + 关键帧网格 + 设置/操作），流程清晰
- **联动**：播放器、关键帧列表、时间轴同步，操作反馈即时
- **可定制**：多种采样策略 + 单帧编辑 + 勾选导出，兼顾自动与人工控制

---

## 四、与本项目的简单对比

| 维度 | 网页版 (video2ppt.com) | 本项目 (本地) |
|------|------------------------|----------------|
| 运行方式 | 在线、上传视频 | 本地运行，YouTube 下载 + 本地提取 |
| 关键帧来源 | 自动场景检测 + 时间间隔 + 手动截帧 | PySceneDetect 场景检测 或 evp 相似度翻页 |
| 采样方式 | 间隔 1s/5s/10s、自定义、智能人像 | 阈值、最小场景、静态/重复过滤 |
| 后期编辑 | 单帧裁剪、删除、勾选导出 | 目前无单帧编辑，可考虑后续增加 |
| 导出格式 | 图片 / PDF / PPTX / 分享 | PDF、PPTX、单页图片 |
| 手动补帧 | 支持按「C」或时间轴截帧 | 暂无 |
